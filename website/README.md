# Paper Dates documentation

Documentation site for [react-native-paper-dates](https://github.com/web-ridge/react-native-paper-dates), built with [Rspress](https://rspress.rs/).

## Setup

```bash
yarn install
```

## Development

```bash
yarn dev
```

## Build

```bash
yarn build
```

## Preview

```bash
yarn preview
```

The production site is deployed to GitHub Pages at `https://web-ridge.github.io/react-native-paper-dates/`.

[React Native Paper Dates](https://github.com/web-ridge/react-native-paper-dates) is a cross-platform [Material Design](https://m3.material.io/) date and time picker library for [React Native Paper](https://reactnativepaper.com/).

## What's included

- [Date pickers](./date-picker/single-date-picker) — single, range, multiple, and input modes
- [Time picker](./time-picker) — clock face or keyboard input
- [Localization](./localization) — built-in translations plus custom and dynamic locales

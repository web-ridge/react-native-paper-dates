    #Introduction

    React Native Paper Dates is a cross-platform Material Design date and time picker library for React Native Paper.

    #What's included

    • Date pickers — single, range, multiple, and input modes
    • Time picker — clock face or keyboard input
    • Localization — built-in translations plus custom and dynamic locales

    #Next steps

    1. Install the library
    2. Register a translation
    3. Pick a component from the sidebar — each page has a live preview so you can see the UI immediately

    PaperDates

    Material Design 3 date & time pickers for React Native — on Android, iOS, and web.

    Get startedView on GitHub

    Add it to your project

    Drop in beside React Native Paper. One package, every platform.

    $ yarn add react-native-paper-dates

    Picker modes

    Every selection pattern, one Material language

    1. 01Single dateOne day, calendar or input — matching Material date pickers.→
    2. 02Date rangeStart and end selection with tonal range highlighting.→
    3. 03Multiple datesToggle many days in a single Material flow.→
    4. 04Input dateText field with calendar affordance for typed or picked dates.→
    5. 05Time pickerClock and input modes aligned with Material time pickers.→
